# Contexto do Projeto: Análise de Ideias de Micro SaaS

## Objetivo Principal
Analisar opções para desenvolvimento de Soluções de Micro SaaS

## Restrições e Diretrizes
*   **Hardware:** MacBook 2012 com macOS Catalina 10.15.7. As soluções devem ser leves e performáticas.
*   **Tecnologias:**
    *   **Backend:** .NET Core
    *   **Frontend:** React, Angular ou Vue.js
    *   **Banco de Dados:** PostgreSQL ou MySQL
    *   **Software Instalado:** Node.js, .NET Core, Multipass, PostgreSQL, MySQL, Python, Pip, Brew.
*   **Ambiente de Produção (Simulado):** Utilizar Multipass para criar e gerenciar uma VM Linux.
*   **Gerenciamento de Estado:** Este arquivo (`Context.md`) será usado para rastrear o progresso e os próximos passos.
