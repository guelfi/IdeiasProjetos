# Análise de Ideias de Micro SaaS

Este repositório contém uma análise detalhada de diversas ideias de Micro SaaS, com foco em viabilidade técnica, mercado e potencial de sucesso. O objetivo é documentar o processo de ideação e planejamento para futuros projetos.

## Conteúdo

-   **`ideias.html`**: Arquivo principal contendo a análise de cada ideia de Micro SaaS, incluindo:
    -   Descrição da ideia
    -   Requisitos técnicos (com foco em ASP.NET Core, C#, PostgreSQL, React Native, Docker, etc.)
    -   Análise de mercado e concorrência
    -   Potencial de sucesso
    -   Prompts para ferramentas de IA (como Gemini CLI e ChatGPT)
    -   Sugestões de nomes para cada projeto

## Restrições e Diretrizes Técnicas

Os projetos são planejados considerando as seguintes restrições e diretrizes técnicas:

-   **Hardware:** MacBook 2012 com macOS Catalina 10.15.7 (soluções leves e performáticas).
-   **Tecnologias:**
    -   **Back-End:** ASP.NET Core 8 LTS, C#
    -   **Banco de Dados:** PostgreSQL, Entity Framework Core
    -   **Autenticação:** JWT (JSON Web Token)
    -   **Arquitetura:** DDD, SOLID, Clean Architecture, Repository Pattern
    -   **Frontend Web (Dashboard):** Razor Pages (foco inicial), possível migração para Blazor Server
    -   **Frontend Mobile:** React Native (Mobile First)
    -   **DevOps:** Docker, Docker Compose, Git, GitHub, Multipass (simulação)
    -   **Google AI & Cloud:** Gemini CLI (Google) e ChatGPT
    -   **Editor:** VS Code (configurado no macOS Catalina)
    -   **Auxiliares:** OpenAI CLI, Azure Data Studio, Node.js (CLI tooling)
    -   **Logging:** Serilog
-   **Ambiente de Produção (Simulado):** Utilização de Multipass para criar e gerenciar VMs Linux.

## Como Usar

Abra o arquivo `ideias.html` em seu navegador para visualizar a análise completa das ideias.
